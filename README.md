A monotonic array is an array that is either:

Monotone increasing → each element is greater than or equal to the previous one.
Example: [1, 2, 2, 3, 5]

Monotone decreasing → each element is less than or equal to the previous one.
Example: [9, 7, 7, 4, 2]
